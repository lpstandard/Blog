command_line

Some basic yet useflul command to play around in your terminal:

  • cd
    • If you wants to go to a folder, just cd [foldername]
  • ls
    • Check all the files in your folder. Ex. $ ls desktop
  • rm
    • Remove a file in your folder.
  • mv

    • Move or rename a file. Ex: 1. mv foo.md bar.md => The name of foo.md is now bar.md
